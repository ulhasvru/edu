# 📚 EduQuest – Class 9 Maharashtra

A fully interactive quiz website for **Class 9 students** from **Jalgaon, Maharashtra** (Maharashtra State Board – Semi-English medium).

## 🌐 Live Website
👉 **[Click here to open EduQuest](https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/)**
> Replace `YOUR-USERNAME` and `YOUR-REPO-NAME` after deploying.

---

## ✨ Features

- 🔐 **Login / Sign Up** – LocalStorage-based authentication
- 📚 **8 Subjects** – Maths, Science, English, Marathi, Hindi, History, Geography, Civics
- 🎯 **30 Questions per Subject** – MCQ, True/False, Fill in the Blank
- 🟢🟡🔴 **3 Difficulty Levels** – Easy, Medium, Hard
- 📊 **Dashboard** – Points, streaks, subject-wise progress
- 🏆 **16 Badges & Awards** – Unlock as you achieve milestones
- 🏅 **Leaderboard** – See your rank among classmates
- 💾 **All data saved in browser** (LocalStorage – no server needed)
- 🎊 **Confetti animations** for great scores
- 📱 **Fully Responsive** – Works on mobile, tablet, desktop

---

## 🚀 How to Deploy on GitHub Pages

1. Create a new repository on [github.com](https://github.com)
2. Upload `index.html` and `README.md`
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch and `/ (root)` folder
5. Click **Save** — your site will be live in ~1 minute!
